# spring-todo-list-bot

Простой телеграмм-бот для составления TODO-списков.

## Возможности

- Добавление новых списков
- Отметка выполненных пунктов в списке
- Удаление ненужных списков

## Стек технологий
- Java 11
- Spring Boot + Spring Data MongoDB
- Telegram Bots Java Library
- Lombok
- Maven
- MongoDB
- Docker + Docker-Compose
